# План: Новый сайт Cosmo Capital

**Дата старта:** 2026-05-21
**Продукт:** cosmacapital.ru — агентство коммерческой недвижимости
**Цель:** Каталог объектов (аренда/продажа) + поиск/фильтры + встроенная мини-CRM

---

## Стек

| Слой | Технология | Обоснование |
|------|-----------|-------------|
| Framework | Next.js 15 (App Router) + TypeScript | уже настроен |
| Styles | Tailwind CSS v4 | уже настроен |
| БД | **Supabase** (PostgreSQL) | хостинг + storage для фото + auth + дашборд |
| ORM | **Prisma** | типизированные запросы, миграции |
| Auth (admin) | **NextAuth.js** + credentials | email/пароль для 1 владельца |
| Хранение фото | **Supabase Storage** | S3-совместимое, бесплатный тир |
| Deploy | **Vercel** | нативная поддержка Next.js |

---

## Схема базы данных

### Таблица `objects` — объекты недвижимости

| Поле | Тип | Описание |
|------|-----|---------|
| id | uuid | PK |
| type | enum(RENT, SALE) | тип сделки |
| category | enum(OFFICE, RETAIL, WAREHOUSE, FREE_PURPOSE, PRODUCTION) | категория |
| title | text | название объекта |
| description | text | полное описание |
| address | text | адрес |
| metro | text? | ближайшее метро |
| area_total | float | общая площадь, кв.м |
| area_min | float? | минимальная секция (для аренды) |
| floor | int? | этаж |
| floors_total | int? | этажей в здании |
| price | float | цена (руб/мес для аренды, руб для продажи) |
| price_per_sqm | float? | цена за кв.м (автовычисляется) |
| photos | text[] | массив URL из Supabase Storage |
| status | enum(ACTIVE, ARCHIVED) | статус публикации |
| featured | bool | показывать в топе |
| created_at | timestamp | |
| updated_at | timestamp | |

### Таблица `leads` — заявки (мини-CRM)

| Поле | Тип | Описание |
|------|-----|---------|
| id | uuid | PK |
| name | text | имя клиента |
| phone | text | телефон |
| email | text? | email |
| message | text? | сообщение |
| object_id | uuid? | FK -> objects (с какого объекта заявка) |
| source | text | "catalog" / "object_page" / "hero" |
| status | enum(NEW, IN_WORK, DONE, REJECTED) | статус обработки |
| notes | text? | заметки менеджера |
| created_at | timestamp | |

### Таблица `services` — услуги компании

| Поле | Тип | Описание |
|------|-----|---------|
| id | uuid | PK |
| title | text | название услуги |
| description | text | описание |
| icon | text? | emoji или имя иконки |
| order | int | порядок отображения |
| active | bool | показывать или нет |
